# SGF UI Skeleton (Fase 1)
UI en HTML/CSS/JS sin funcionalidad de datos.
- Login/Gate
- Navegación por módulos
- Vistas y tablas placeholder
- Modals para todos los botones principales
- JS separado por módulo (carpeta `js/modules/`)
- Preparado para integrar IndexedDB en fase 2

## Ejecutar
Abrí `index.html` en tu navegador.

> Nota: por usar `type="module"`, algunos navegadores requieren servirlo por HTTP.
> Si ves error CORS/Modules, corré un servidor local:
- `python -m http.server 8080` y abrir `http://localhost:8080/`
